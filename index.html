最快开始路径

  1. 需要确定题目：打开选题 Prompt 库，先用 Prompt 1.1。
  2. 题目已经确定：打开 AI 直接找论文的 Prompt，用方案八快速拉清单。
  3. 准备写综述：打开文献综述库，从框架设计开始。
  4. 进入论文成稿：打开顶刊写作库，先做创新性和结构检查。

使用边界：AI 可以加速生成方向、清单和结构，但论文信息、引用、数据和关键判断仍然需要人工验证。

页面边界：本页是静态入口，不修改原 Prompt 内容，不引入构建工具，不依赖外部脚本。
